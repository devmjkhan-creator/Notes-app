# 📝 Notes Simple Task Manager

A simple **Next.js** practice project to manage tasks. Users can easily add, remove, and view tasks. Perfect for beginners to learn **React** and **Next.js** in a hands-on way.

[🌐 Live Demo](#https://notes-app-nine-gold.vercel.app/)  

## ⚡ Features
. ✅ Add tasks instantly  
. ❌ Remove tasks with a click  
. 🖥️ Responsive and clean UI  
. ⚡ Fast, client-side app  
-. 💡 Practice project for Next.js beginners  

## 🛠 Tech Stack
. **Next.js**  React framework for server-side rendering and routing  
. **React**  UI components and state management  
. **CSS / Tailwind CSS / Styled Components**  Styling (adjust based on your project)  
. **Vercel / Netlify**  Optional for deployment  

## 🚀 Getting Started

Follow these steps to run the project locally:

# Clone the repository
git clone https://github.com/devmjkhan-creator/Notes-app.git

# Navigate to the project folder
cd  Notes-app

# Install dependencies
npm install

# Run the development server
npm run dev
